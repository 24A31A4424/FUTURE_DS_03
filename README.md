📊 Marketing Funnel & Conversion Performance Analysis
🚀 Project Overview

This project analyzes a real-world bank marketing dataset to understand customer conversion behavior across a marketing funnel. The goal is to identify drop-offs, channel performance, customer patterns, and conversion drivers to improve lead-to-customer conversion rates.

The analysis includes:

Funnel performance tracking
Conversion rate analysis
Channel effectiveness
Customer segmentation insights
Visual dashboards with multiple charts
📁 Dataset

The dataset used is the Bank Marketing Dataset:

File: bank-full.csv
Source: UCI Machine Learning Repository
Type: Marketing campaign data from a Portuguese banking institution
Key Features:
Age, job, marital status, education
Contact type (cellular, telephone)
Campaign duration
Loan status (housing, personal loan)
Target variable: y (subscription yes/no)
🎯 Objective

To analyze the marketing funnel and answer:

Where do customers drop off in the funnel?
Which channels perform best?
What customer segments convert better?
What factors influence conversion?

📊 Key Insights
📌 Cellular contact performs better than telephone
📌 Call duration strongly impacts conversion
📌 Certain job roles (e.g., students, retired) convert more
📌 Longer engagement leads to higher conversion probability
📌 Customers with no housing loan convert more often
📌 Seasonal trends affect campaign success
📉 Marketing Funnel Insight

The funnel shows significant drop-off at the engagement stage, meaning:

Many leads are contacted, but fewer are properly engaged.

💡 Business Recommendations
🎯 Improve Conversion Rate
Focus on increasing engagement time
Train agents for better call quality
📞 Optimize Channels
Increase investment in cellular outreach
Reduce low-performing contact methods
👥 Customer Targeting
Focus on high-converting job segments
Use age-based targeting strategies
📅 Timing Strategy
Run campaigns during high-performing months (Mar, Sep, Oct)
